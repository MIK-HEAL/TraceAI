# TraceAI v0.1.0-beta.1

TraceAI 的首个试用版已经可以安装和验证。

## 你会得到什么

- 统一事件模型
- SQLite 本地存储
- 事件总线
- MCP / OpenAI 接入
- CLI 统计、导出、报表
- `status` / `health` / `metrics`
- Dashboard 只读查看

## 安装

```bash
go install github.com/MIK-HEAL/TraceAI/cmd/traceai@v0.1.0-beta.1
```

## 快速验证

```bash
traceai version
traceai health
traceai metrics --format json
```

## 适用场景

- 本地试用
- 早期接入验证
- MCP / Agent 行为分析

## 不建议

- 作为唯一生产监控系统
- 直接替代成熟平台
